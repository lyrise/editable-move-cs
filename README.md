# VRename

[![Nuget](https://img.shields.io/nuget/v/VRename)](https://www.nuget.org/packages/VRename)

## Install

```sh
dotnet tool install --global VRename
```

## Usage

### Find

```cs
vrename find -d ./ -p "**"
```

### Run

```cs
vrename run
```

### Undo

```cs
vrename undo
```

### Clean

```cs
vrename clean
```
