# emv

[![Nuget](https://img.shields.io/nuget/v/emv)](https://www.nuget.org/packages/emv)

## Install

```sh
dotnet tool install --global editable-move
```

## Usage

### Find

```cs
emv find -d ./ -p "**"
```

### Run

```cs
emv run
```

### Undo

```cs
emv undo
```

### Clean

```cs
emv clean
```
