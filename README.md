# VRename

[![Nuget (with prereleases)](https://img.shields.io/nuget/vpre/VRename)](https://www.nuget.org/packages/VRename)

## Install

```sh
dotnet tool install --global VRename --version 0.0.1-alpha.1
```

## Usage

### Find

```cs
vrename find -d ./ -p "**"
```

### Run

```cs
vrename run
```

### Undo

```cs
vrename undo
```

### Clean

```cs
vrename clean
```
